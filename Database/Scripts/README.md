# Database Scripts

This folder will contain SQL scripts for creating and managing the database.

## Scripts Structure

- `01_CreateDatabase.sql` - Database creation script
- `02_CreateTables.sql` - Table creation scripts
- `03_SeedData.sql` - Initial data seeding
- `04_StoredProcedures.sql` - Stored procedures
- `05_Views.sql` - Database views

## Usage

Scripts will be executed in order to set up the complete database schema.
